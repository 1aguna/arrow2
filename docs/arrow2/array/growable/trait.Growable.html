A trait describing a struct that can be extended from slices of pre-existing Arrays. This is used in operations where a new array is built out of other arrays such, as filtering and concatenation.

Required methods

fn extend(&mut self, index: usize, start: usize, len: usize)[src]

Extends this Growable with elements from the bounded Array at index index from a slice starting at start and length len.

Panic

This function panics if the range is out of bounds, i.e. if start + len >= array.len().
