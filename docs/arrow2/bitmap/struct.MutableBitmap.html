pub struct MutableBitmap { /* fields omitted */ }

A container to store booleans. MutableBitmap is semantically equivalent to Vec<bool>, but each value is stored as a single bit, thereby achieving a compression of 8x. This container is the counterpart of MutableBuffer for boolean values. MutableBitmap can be converted to a Bitmap at O(1).

Implementation

This container is backed by MutableBuffer<u8>.
